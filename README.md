# Z3 Extension for Visual Studio Code

## Features

- Syntax highlighting for SMT-LIB2 files (`.smt2`, `.smt`) from [SMT.tmbundle](https://github.com/SRI-CSL/SMT.tmbundle)
- Execute SMt-LIB2 files with Z3 without explicitly installing Z3

## Usage

For executing SMT-LIB2 files, open a `.smt2` or `.smt` file and click on the `â–¶ Execute` button in the top right corner of the editor. The result will be displayed in the output window. You can also open the command palette (Ctrl+Shift+P) and type `Z3: Execute`.
